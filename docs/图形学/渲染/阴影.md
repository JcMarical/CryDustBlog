# 软阴影 
## PCF 百分比
取深度，获取周围n个点，求个visiability = 比当前深度值大的数/n。

## PCSS
阴影越靠近投射物，阴影越硬
* 半影区域：物体到阴影平面的距离，决定阴影的软硬程度（采样半径R的大小）
## VSSM
* 试图使用计算的方式避免采样（太慢）
* 如果我们能够获取到这个百分比，就不需要采样了
* 切比学府不等式估计（方差）
## SDF 阴影
光线步进，逐步递进判断是否打到物体了
