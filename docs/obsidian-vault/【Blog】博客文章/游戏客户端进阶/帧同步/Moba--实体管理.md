# Moba--实体管理

# 塔
只需要定义塔配置id以及出生位置即可。

各个单位配置搭上自己的技能ID。

## 生成时机
在战斗初始化时生成即可。

## 逻辑执行
也是由战斗类的LogicTick来负责塔的LogicTick。
具体主要功能：
- 定时检测配置中的目标，如果存在目标则发射目标跟踪技能。

## 表现执行
逻辑驱动表现：
主要做的是
- 场景中实体的管理与摧毁。
- 音效管理
- 动画管理
- UI管理。
# 小兵
定义小兵配置id、出生位置、出生延迟、各个小兵出生间隔、以及波次间隔。

也是各个单位配置搭上自己的技能id 

## 生成时机
计时器，由塔生出。
