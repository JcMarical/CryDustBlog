# 一 热补丁的作用
如果游戏逻辑都是通过C#写的，后期想要热更新相当的麻烦。
但是通过热补丁，我们可以实现不走原C#中的逻辑，转而去执行Lua中的代码。

必须做四个非常重要的操作
- 1.加特性
- 2.加宏
- 3.生成代码
