# 二叉树

> 核心思路：
> 1.分解问题：动规--子树（整块，一般需要返回值或辅助结构）
> 2.遍历：回溯-树枝（移动的过程），DFS-节点（单一处理）
> 前序：只能拿到参数父节点数据
> 中序：参数与左返回值，一般解决==BST问题==
> 后序：参数，左右返回值，**效率最高**。关键词：==子树，含返回值==
> （直径问题-树形DP，前缀和）
